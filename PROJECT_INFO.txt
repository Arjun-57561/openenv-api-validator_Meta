================================================================================
PROJECT OVERVIEW
================================================================================

This is an OpenEnv-compatible API Response Validator project designed for the
Meta x PyTorch x Scaler hackathon. The system validates HTTP API responses
against contracts, schema risks, and documentation using a combination of
rule-based and LLM-based scoring.

Key Components:
- FastAPI server for environment endpoints
- Multi-level difficulty validation (easy, medium, hard)
- Rule-based and LLM-powered grading
- Docker containerization for deployment
- OpenEnv specification for standardization
